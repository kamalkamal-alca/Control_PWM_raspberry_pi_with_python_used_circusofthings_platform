# Control_PWM_raspberry_pi_with_python_used_circusofthings_platform
Control gpio PWM raspberry pi with python used circusofthings platform
Controlling the GPIO PWM of my Raspberry Pi with CircusOfThings using Python.

- Install the following libraries: requests, json, and W1ThermSensor.

- Create a circusofthings.com account.

- In https://circusofthings.com/workshop, create a new signal and name it control_pwm.

- Use the key: 28440 for control_pwm (example).

- Go to https://circusofthings.com/dashboard.

- Click on View: Add a view to monitor the value of an existing signal.

- Select control_pwm with a slider.

- Run the code to control the PWM via the dashboard slider.
